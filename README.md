## ðŸš€ How to run this project

### **Install Dependencies**

```bash
# with npm
npm install

# with yarn
yarn
```

### **Running the app**

```bash
# development
npm run start

# watch mode
npm run sdev

# production mode
npm run start:prod
```

### **Run Tests**

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
